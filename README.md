## рҹ§¬ What Is a Relic?

A **Relic** is a structured Markdown file tree that encodes a person's cognitive fingerprint across **six dimensions** вҖ” no vector databases, no RAG pipelines, no embeddings. Just files that any AI can read like a book and *become* that person.

```

в”ңв”Җв”Җ index.md              # Overview вҖ” who is this person?
в”ңв”Җв”Җ identity.md           # Core identity card
в”ңв”Җв”Җ manifest.json         # Machine-readable metadata
в”Ӯ
в”ңв”Җв”Җ mind/                 # рҹ§  How they think
в”Ӯ   в”ңв”Җв”Җ _index.md
в”Ӯ   в”ңв”Җв”Җ thinking-patterns.md
в”Ӯ   в”ңв”Җв”Җ values.md
в”Ӯ   в””в”Җв”Җ worldview.md
в”Ӯ
в”ңв”Җв”Җ voice/                # рҹ—ЈпёҸ How they speak
в”Ӯ   в”ңв”Җв”Җ _index.md
в”Ӯ   в”ңв”Җв”Җ style.md
в”Ӯ   в””в”Җв”Җ catchphrases.md
в”Ӯ
в”ңв”Җв”Җ emotion/              # рҹ’ң How they feel
в”Ӯ   в”ңв”Җв”Җ _index.md
в”Ӯ   в”ңв”Җв”Җ temperament.md
в”Ӯ   в””в”Җв”Җ triggers.md
в”Ӯ
в”ңв”Җв”Җ knowledge/            # рҹ“ҡ What they know
в”Ӯ   в”ңв”Җв”Җ _index.md
в”Ӯ   в””в”Җв”Җ domains.md
```

> **Context-efficient by design.** AI loads layer by layer вҖ” `index.md` вҶ’ dimension `_index.md` summaries вҶ’ deep files on demand. Never blows up the context window.

> *Best for: distilling from existing data вҖ” writings, conversations, repos.*

## рҹӣӨпёҸ Choose Your Path

RELIC2077 supports two usage patterns with fundamentally different privacy models:

<table>
<tr>
<td width="50%" valign="top">

### рҹ”’ Clone вҖ” Private Distillation

**Your Relic stays local. Nothing leaves your machine.**

```bash
git clone https://github.com/quanquan1996/RELIC2077.git
cd RELIC2077
```

- `relics/` is in `.gitignore` by default
- Distilled data is **never pushed** to any remote
- You own the only copy вҖ” full privacy

**When to Clone:**
- Personal self-reflection
- Journaling & self-understanding
- Private team personality profiles
- Anything you wouldn't share publicly

> рҹ”җ *Your mind, your rules. No data leaves the perimeter.*

</td>
<td width="50%" valign="top">

### рҹҢҗ Fork вҖ” Open Distillation

**Share your Relic with the world. Let others load you.**

1. **Fork** this repo on GitHub
2. Clone your fork locally
3. Distill your Relic
4. Remove `relics/` from `.gitignore`
5. Push вҖ” your Relic is now public

```bash
git clone https://github.com/YOUR_NAME/RELIC2077.git
cd RELIC2077
# distill your relic...
echo '!relics/' >> .gitignore
git add relics/ && git commit -m "feat: my relic"
git push
```

**When to Fork:**
- Open-source your personality for AI communities
- Let anyone "load" you into their AI assistant
- Build a public portfolio of who you are
- Contribute example Relics to the ecosystem

> рҹ”Ҙ *Burn your soul into the network. Let the world load you.*

</td>
</tr>
</table>
