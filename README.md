## рҹ§¬ What Is a Relic?

A **Relic** is a structured file tree that encodes a person's essence across six dimensions вҖ” no vector databases, no RAG pipelines. Just pure Markdown files that any AI can read like a book.

```
johnny-silverhand/
в”ңв”Җв”Җ index.md              # Who is this person?
в”ңв”Җв”Җ identity.md           # Core identity card
в”ңв”Җв”Җ manifest.json         # Machine-readable metadata
в”ңв”Җв”Җ mind/                 # рҹ§  How they think
в”ңв”Җв”Җ voice/                # рҹ—ЈпёҸ How they speak
в”ңв”Җв”Җ emotion/              # рҹ’ң How they feel
в”ңв”Җв”Җ knowledge/            # рҹ“ҡ What they know
в”ңв”Җв”Җ relations/            # рҹӨқ How they connect
в””в”Җв”Җ {custom}/             # вҡЎ Elastic extension
```

> An AI loads a Relic layer by layer вҖ” `index.md` first, then `_index.md` summaries, then deep files on demand. Context-efficient. Never blows up the window.

## вҡЎ The Six Dimensions

| Dimension | Dir | What It Captures |
|:-:|:-:|---|
| рҹ§  **Mind** | `mind/` | Thinking patterns, values, worldview, blind spots |
| рҹ—ЈпёҸ **Voice** | `voice/` | Language style, catchphrases, humor, register shifts |
| рҹ’ң **Emotion** | `emotion/` | Temperament, triggers, coping patterns |
| рҹ“ҡ **Knowledge** | `knowledge/` | Expert domains, depth of knowledge |
| рҹӨқ **Relations** | `relations/` | Social style, trust patterns, group dynamics |
| вҡЎ **Custom** | `{name}/` | Elastic вҖ” anything else that defines them |

## рҹ”® Two Modes of Distillation

### рҹ’¬ Interactive Mode
The AI interviews you вҖ” structured conversation across all six dimensions. It writes the Relic as you talk.

> *Best for: distilling yourself or someone you're interviewing live.*

### вҡҷпёҸ Batch Mode
Point at a directory вҖ” chat logs, blog posts, code repos, notes вҖ” and a multi-agent pipeline extracts the Relic automatically.

```
Orchestrator scans files
  вҶ’ 5 Dimension Extractors run in parallel
    вҶ’ Synthesizer merges, validates, generates index
      вҶ’ Dynamic fission splits oversized files
        вҶ’ вң… Relic complete
```

> *Best for: distilling from existing data вҖ” writings, conversations, repos.*
